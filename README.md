### Dependencies

node v16.13.2
npm 8.1.2
python v3.8.10
pip v20.0.2

### Local python setup (optional)

```
cd <project root>
python -m venv venv
source venv/bin/activate
pip install -r backend/requirements.txt
```

### Setup frontend 

```
cd frontend;
npm install
```

### backend 
```
command to run backend: python3 manage.py runserver 0.0.0.0:8000
```
